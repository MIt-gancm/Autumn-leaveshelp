## 目录完整

::: file-tree
- ~ **HOME** 目录
  - .gancm **脚本** 目录
    - .git 无需理会
    - config 配置文件
    - function 部分功能
    - local 主要功能文件夹
    - gancm.sh 脚本入口
    - README.md 说明文档
    - log.log 日志调用了脚本什么功能都在这
  - .termux **自带的配置** 目录
    - gancm 
      - proot 
        - 你的容器
          - root
            - .gancm 回到开头
      - MCserver Linux mc服务器文件夹后续会更改位置
        - 服务器核心
          - 服务器版本
            - 服务器核心
            - start.sh 启动脚本 MC服务器无法启动发这个截图
- …
:::
